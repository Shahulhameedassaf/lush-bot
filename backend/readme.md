## 🚀 Quick Start
Before running LushBot, install these tools:

1. **Docker Desktop**  
   Download and install: [https://www.docker.com/products/docker-desktop/](https://www.docker.com/products/docker-desktop/)
   - Windows: Enable WSL 2 during installation
   - Mac/Linux: Follow default installation

2. **Ollama (LLM Runtime)**  
   Download and install: [https://ollama.com/download](https://ollama.com/download)
   - Installs locally on your machine
   - No account required

3. **Python 3.9 or higher**  
   Download: [https://www.python.org/downloads/](https://www.python.org/downloads/)
   - During installation, check "Add Python to PATH"

4. **Pull Llama Model**  
   After installing Ollama, open terminal and run:
   ```bash
   ollama pull llama3.2:3b
