# LushBot - Setup & Run Guide

If you just downloaded this code from Git, follow these steps to get it running.

## 1. Install Prerequisites
You need these installed on your computer:
1.  **Python 3.10+**: [Download Here](https://www.python.org/downloads/) (Check "Add Python to PATH" during install)
2.  **Docker Desktop**: [Download Here](https://www.docker.com/products/docker-desktop/) (Required for the database)
3.  **Ollama**: [Download Here](https://ollama.com/) (Required for AI models)

## 2. Setting Up Environment and Requirements
1.  **Open Terminal** (Command Prompt or PowerShell) at the project's root folder.
2.  **Create a Virtual Environment** (Required for the Starter Script):
    Run this command:
    ```bash
    python -m venv .venv
    ```
3.  **Download the AI Model**:
    Run this command:
    ```bash
    ollama pull llama3.2:3b
    ```
4.  **Install Python Libraries**:
    Navigate to the `backend` folder and run this command:
    ```bash
    cd backend
    ..\\.venv\\Scripts\\python -m pip install -r requirement.txt
    ```
    *(This ensures libraries are installed specifically for this project)*

5.  **Initial Data Ingestion** (Load the resort info):
    While still in the `backend` folder, run:
    ```bash
    ..\\.venv\\Scripts\\python ingest.py
    ```

## 3. Starting the App
You can start everything with one click!

1.  **Run the Starter Script**:
    -   Go to the main folder.
    -   Double-click **`START_LUSHBOT.bat`**.

    *This script will automatically:*
    *   Start the Database (Docker/Qdrant)
    *   Start the AI Engine (Ollama)
    *   Start the Python Backend
    *   Open your browser to the App
